Average House Price Over 20 years Period For All Regions In Uk

The data provided by ONS had to be manually aggregated in order to provide user with user friendly visual approach. The intial raw data, contain many irrelevant and unnecessary data, plus the partation of the average all dweeling price was split into quatars, which was aggregated to find the average for the whole year. This will provide user with easier snapshot and readability of the data in a line graph. The cleaned up data could be found in the github, which was used to read the data in our code.

In order to successfully implement the line graph and to provide user with the interactivity, multiple existing d3 visualisation implementation were used to successfully build this graph. One of the key concept to create this graph was to aim for simplicity and accessibility, so the users are not overwhelmed with too many information and is also suitable for colour-blind people.
